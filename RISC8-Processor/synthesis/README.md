# Synthesis Results - 8-bit RISC Processor

## 📌 Overview

This folder contains the synthesis results of the 8-bit RISC Processor generated using Xilinx Vivado. Synthesis converts the Verilog HDL design into a gate-level hardware implementation suitable for FPGA deployment.

---

## 📂 Folder Contents

| File | Description |
|------|-------------|
| rtl_schematic.png | RTL schematic generated after synthesis |
| utilization_report.png | FPGA resource utilization report (if available) |
| timing_report.txt | Timing analysis report (if available) |
| README.md | Synthesis documentation |

---

## 🛠 Synthesis Environment

- Tool: Xilinx Vivado
- HDL Language: Verilog
- Target Device: FPGA

---

## ▶ Synthesis Procedure

1. Create a Vivado project.
2. Add all RTL files.
3. Set `risc8_top` as the top module.
4. Run **Run Synthesis**.
5. View RTL schematic and synthesis reports.
