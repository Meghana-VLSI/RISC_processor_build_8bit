## 🧩 Main Components

- Program Counter (PC)
- Instruction Memory
- Control Unit
- Register File
- Arithmetic Logic Unit (ALU)
- Data Memory
- Write Back Unit

---


## 🔄 Working

1. Fetch instruction.
2. Decode instruction.
3. Read operands.
4. Execute ALU operation.
5. Access memory (LOAD/STORE).
6. Write result back to register.
