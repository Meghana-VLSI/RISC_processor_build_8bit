# Documentation

## Overview

This folder contains the complete documentation for the **8-Bit RISC Processor** project developed using **Verilog HDL** and **Xilinx Vivado Design Suite**.

The documentation provides a detailed explanation of the processor architecture, design methodology, implementation, simulation, synthesis, and verification results. It serves as a reference for understanding the project and its development process.

## Contents

- Project Report
- Architecture Description
- Block Diagram
- State Diagram
- Instruction Set
- Verilog Module Description
- Simulation Results
- RTL Schematic
- Synthesis Results
- Testbench Description
- Conclusions and Future Scope

## Software Used

- Verilog HDL
- Xilinx Vivado Design Suite

## Purpose

The documentation helps users understand the internal working of the 8-Bit RISC Processor, including instruction execution, processor architecture, hardware implementation, and functional verification.
